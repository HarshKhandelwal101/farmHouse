# farmHouse
A small web app to get familiar with Node.js.

# Learnings
1. Using core modules such as http,url and fs.
2. Synchronous and Asynchronous nature of Node.js.
3. Creating a simple web server.
4. Building APIs.
5. HTML templating.
6. Using User-defined modules.
